# Prerequisites
* Yarn
* npm

# Set-up
Go to
````
cd Hotel-Booking-WebApp-main/src/lib
````
Open the file
````
firebase.js
````
And add your Firebase Config details

# To execute
Go to
````
cd Hotel-Booking-WebApp-main
````
````
yarn run dev
````

# Built using

1. React.js
2. Firebase
3. Material UI
